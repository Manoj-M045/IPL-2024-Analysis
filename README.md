# IPL 2024 Data Analysis

## Project Overview
This project analyzes the **IPL 2024** season using Python, Pandas, Matplotlib, and Seaborn. It explores player performances, team statistics, and match trends through meaningful data visualizations.

## Objectives
- Analyze IPL 2024 match and ball-by-ball data.
- Identify top-performing batsmen and bowlers.
- Compare team performances and partnerships.
- Visualize important IPL statistics.

## Visualizations
- Orange Cap (Top Run Scorers)
- Most Sixes
- Super Striker
- Purple Cap (Top Wicket Takers)
- Green Dot Balls
- Top Opening Partnerships
- Team vs Opponent Wins (Heatmap)
- Team vs Venue Average Score (Heatmap)
- Team-wise Match Scores (Boxplot)

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Dataset
- `matches_2024.csv`
- `deliveries_2024.csv`

## Results
The analysis provides valuable insights into player performances, team strengths, scoring patterns, and venue-based trends through nine different visualizations.

## Conclusion
This project analyzes the IPL 2024 season using Python, Pandas, Matplotlib, and Seaborn to uncover insights from match and ball-by-ball data. Through visualizations, it highlights player performances, team statistics, and overall season trends in a clear and meaningful way.
